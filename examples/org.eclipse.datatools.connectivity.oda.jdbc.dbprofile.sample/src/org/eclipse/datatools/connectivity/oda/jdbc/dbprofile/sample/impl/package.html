This package provides an exemplary implementation of an ODA driver's runtime extensions that adopts the integrated ODA Database Connection Profile framework. See the extensions defined in the plugin.xml. It adopts the ODA framework to define an ODA data source type that links to a DTP Database (JDBC) connection profile.

Its scope focus on supporting the design of an ODA data source connection. It uses the auto-generated runtime classes of an ODA driver, included for demo purpose only.

@since DTP 1.6
