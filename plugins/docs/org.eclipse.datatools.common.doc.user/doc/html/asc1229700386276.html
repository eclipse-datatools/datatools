Apache Derby Tutorial Prerequisites

To use the tutorial, you must first install Eclipse, Data Tools Platform, and Apache Derby.

To use the tutorial, install the following:
  • Appropriate versions of Eclipse, Data Tools Platform, EMF, and GEF

    Download or access these builds from https://www.eclipse.org/datatools/downloads.php

  • Apache Derby

    Download the appropriate version from http://db.apache.org/derby/

You should be familiar with basic Eclipse features, for example, perspectives, views, editors, and preference pages, which are used by the tutorial. See the Eclipse Workbench User Guide on the Eclipse online bookshelf, or download it at http://www.eclipse.org/documentation/ for descriptions of these features.
