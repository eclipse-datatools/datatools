[Non-API] Flat File Data Source Designer - an ODA designer UI extension.

Note: The implementation classes are not public APIs.
Backward compatibility support in future releases is not guaranteed.

Package Specification

The ODA flat file designer serves as an exemplary implementation of the ODA designer UI framework and extension points, which integrates with the Data Tools Connection Profile framework.

It allows an user to create and edit an ODA data set design, which defines a CSV data file location and the slice of data to access through its run-time driver. @see org.eclipse.datatools.connectivity.oda.flatfile @since 3.0
